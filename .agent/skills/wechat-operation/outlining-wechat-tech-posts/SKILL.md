# Outlining WeChat Tech Posts

## When to use
大纲、目录、结构重建、教程型骨架（步骤+检查点+报错）。

## Inputs
- 主题
- 内容类型（教程/科普/深度解读/工具评测/复盘/观点）
- 长度目标（默认 1200-2000 字）
- 读者水平（默认中级）

## Output expectations
- 3级标题大纲
- 每节有“读者收益句”
- 必含：误区澄清、踩坑清单、行动清单、30秒速读版要点
- 教程型：前置条件、步骤1-7、检查点、常见失败原因、最终交付物

## Quality checks
- 先结论后细节
- 每节都有“例子/类比/步骤”至少一个落地支撑
