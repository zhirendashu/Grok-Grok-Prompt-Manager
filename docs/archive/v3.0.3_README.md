# v3.0.3 稳定版本说明

## 版本信息
- **版本号**: v3.0.3
- **状态**: 稳定版（Stable Release）
- **创建时间**: 2026-01-12

## 与 v3.0.4 的区别

### ✅ 包含的功能（已验证稳定）
1. **v3.0.2** - 智能修饰词路由
   - 修饰词会根据草稿箱状态智能插入到正确位置
   
2. **v3.0.3** - 面板状态持久化
   - 面板大小、位置、折叠状态会跨会话保存
   - 存储空间预警（5MB阈值）

### ❌ 移除的功能（v3.0.4导致启动失败）
- Robust Prompt Inspector（增强型提示词审查器）
- Fetch Hook 嗅探逻辑
- GLOBAL_POST_STORE 全局存储

## 推荐使用
如果您需要**稳定可靠的脚本**，请使用此 v3.0.3 版本。
v3.0.4 的 Inspector 功能将在修复后重新发布。
