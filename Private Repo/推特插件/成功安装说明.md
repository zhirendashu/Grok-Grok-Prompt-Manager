## 🎮 使用方法

### 方法一：使用 UI 面板（推荐）

1. **点击右下角的🐦图标**
2. **控制面板会弹出**，包含：
   - 📊 统计信息（点赞/关注/转发/下载数量）
   - ⚙️ 功能设置（4个复选框开关）
   - 🎮 控制按钮（启动/停止）
   - 📝 实时日志

3. **勾选需要的功能**：
   - ☑️ 自动点赞
   - ☑️ 自动关注
   - ☑️ 自动转发
   - ☑️ 只看原创

4. **点击"启动"按钮**

5. **查看统计和日志**

### 方法二：使用控制台命令

按 **F12** 打开控制台，输入：

```javascript
// 启用自动点赞
TwitterAssistant.state.autoLike = true;

// 启动自动化
TwitterAssistant.automation.start();

// 查看统计
TwitterAssistant.getInfo();

// 停止自动化
TwitterAssistant.automation.stop();
```

### 方法三：使用右键菜单

1. 点击 Tampermonkey 图标
2. 找到 "Twitter/X 智能助手 v3.0 测试版"
3. 使用右键菜单：
   - 🚀 启动自动化
   - ⏸️ 停止自动化
   - 📊 查看统计
   - 🔄 重置统计

## ⚙️ 功能说明

### 自动化功能
- **自动点赞**：自动为时间线上的推文点赞
- **自动关注**：自动关注推文作者
- **自动转发**：自动转发推文
- **只看原创**：只处理原创推文，跳过转发内容

### 速率限制（每小时）
- 点赞：50 次
- 关注：20 次
- 转发：15 次

### 其他功能
- **高清图片**：自动将图片替换为 4096x4096 高清版本
- **媒体下载**：支持下载图片和视频
- **性能优化**：使用 LRU 缓存和节流优化
