# 🚀 快速上手 - v4.3.0 随机模式

## ⚡ 3分钟开始使用

### 📝 文字模式设置（左侧面板）

1. **创建专用库**：

   ```
   库名：写真模式标准描述
   内容示例：真实胶片直闪摄影，亚洲女性，小红书网红脸，表情冷漠而自信
   ```

2. **使用**：
   - 点击骰子 → 选择"📸 写真模式"
   - 系统自动生成提示词

### 🎬 视频模式设置（右侧面板）

1. **创建专用库**：

   ```
   库名：视频提示词
   内容示例：Fabric: 细致黑色蕾丝边缘动作微微晃动，露出圆润头
   ```

2. **使用**：
   - 点击骰子 → 选择"🎬 视频随机"
   - 系统自动生成提示词

## 📊 所有模式对照表

| 面板 | 模式 | 所需库 | 抽取数量 |
|------|------|--------|---------|
| 文字 | 📸 写真模式 | 写真模式标准描述 | 1+10条 |
| 文字 | 🔞 R18写真 | 写真模式标准描述<br>成人模式标准添加词 | 1+1+10条 |
| 文字 | 🎲 三连抽取 | 当前库 | 3条 |
| 文字 | 🎨 多类混合 | 当前库（跨分类） | 各1条 |
| 文字 | 🌀 混沌生成 | 所有库 | 5-12条 |
| 视频 | 🎬 视频随机 | 视频提示词 | 1条 |
| 视频 | 🔞 R18视频 | 视频提示词<br>R18视频添加提示词 | 1+1条 |

---

## ⚠️ 常见错误

❌ **在左侧面板创建视频库** → 类型标记错误  
✅ 视频库必须在**右侧面板**创建

❌ **库名拼写错误** → 系统找不到库  
✅ 库名必须精确匹配或包含关键词

❌ **视频提示词过多** → AI混淆  
✅ 视频模式只抽1-2条，保持精简
