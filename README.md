## 📖 简介 (Introduction)

**Grok Prompt Manager** 是一款专为 **Grok** 设计的终极提示词管理工具。它不仅仅是一个剪贴板，更是一个集成了 **智能灵感、自动化工作流、美学 UI** 于一体的创作伴侣。

无论你是追求极致画质的 AI 艺术家，还是需要从零开始的初学者，这个脚本都能让你的 Grok 体验提升到一个新的维度。它采用现代化的 **玻璃拟态 (Glassmorphism)** 设计，完美融入 Grok 原生界面，提供丝滑的操作体验。

## ⚡ 快速上手 (Quick Start)

### 📸 3分钟上手写真模式

专为摄影师打造，一键生成高质量人像。

1. **左侧文字面板**：创建专用库
   - **库名**：`写真模式标准描述` (必须精确匹配)
   - **内容建议**：添加你喜欢的通用开头，例如：
     > `真实胶片摄影，时尚人像，自然光，表情生动`

2. **开始使用**：
   - 点击面板下方的 **🎲 骰子图标**
   - 选择 **"📸 写真模式"**
   - 脚本会自动：`取1条你的开头` + `随机混入10条风格修饰词` = **完美提示词**

### 🎬 视频模式防坑指南

视频生成对提示词很敏感，太多词会导致严重幻觉。

1. **右侧视频面板**：创建专用库 (注意是在右侧创建！)
   - **库名**：`视频提示词` (必须精确匹配，或者包含此关键词)
   - **内容建议**：只写动作或环境，每条保持独立。例如：
     > `Cinematic: 晨光通过树叶洒在草地上，微风吹拂，4k画质`

2. **开始使用**：
   - 点击右侧面板的 **🎲 骰子图标**
   - 选择 **"🎬 视频随机"**
   - 脚本只会抽取 **1条** 最精准的描述，避免干扰 AI。

## ✨ 核心亮点 (Highlights)

### 🎨 玻璃拟态美学 UI

摒弃了传统脚本简陋的界面，采用半透明磨砂质感面板，搭配极简线条图标。支持 **拖拽移动、边缘调整大小、自动贴边折叠**。

### 📚 智能双库管理

独创的 **双库分离机制**，告别混乱：

- **🖼️ 图片模式**：自动加载绘画提示词库。
- **🎥 视频模式**：自动切换视频专用提示词库。

### 🎲 灵感随机骰子 (Magic Dice)

卡壳时的救星！

- **📸 写真模式**：一键生成专业人像结构。
- **🔞 R18 模式**：双重库组合（需自行配置）。
- **🌀 混沌模式**：全库大乱炖，寻找意外惊喜。

### 🤖 自动化工作流

- **自动重试**：网络错误自动点击重试，无需人工守候。
- **🎥 批量视频**：想跑一组视频？勾选“一键生成全部”，自动跑完队列。

### 🔍 提示词探测器 (Inspector)

Browsing 时的学习利器。自动在图片卡片上显示 **"✦"** 按钮，点击即可查看：

- **完整提示词 (Full Prompt)**
- **修改参数 (Modifiers)**
- **一键复制/保存**

## 🛠️ 功能详解 (Features)

| 功能模块 | 详细说明 |
| :--- | :--- |
| **库管理** | 支持无限新建库、重命名、拖拽排序、JSON 导入导出。 |
| **粘贴导入** | 支持 `【标题】+内容`、`Markdown`、`Excel` 等多种格式智能解析。 |
| **快捷操作** | 右键菜单支持置顶、删除；双击直接编辑提示词。 |
| **数据安全** | LocalStorage 本地存储，支持配额监控与手动备份。 |
| **视频专用** | 视频面板拥有独立的“精简随机模式”，避免提示词过多产生幻觉。 |

### 📋 智能导入格式支持

直接复制粘贴文本，脚本会自动识别：

```text
【写真标题】
这里是详细的描述内容...

【另一个标题】
这里是另一个描述...
```

## 💡 如何安装与使用

### 安装

1. 安装浏览器扩展 [Tampermonkey](https://www.tampermonkey.net/)。
2. [**点击这里安装脚本**](https://greasyfork.org/) (或手动加载 `01_正式版` 中的文件)。
3. 打开 [Grok.com](https://grok.com/)，刷新页面。

### 使用

1. **唤起面板**：点击屏幕侧边半透明的 **G 图标** 或 **🤖 机器人图标**。
2. **切换模式**：左侧面板管理图片，右侧面板管理视频（自动/手动切换）。
3. **随机灵感**：点击面板下方的 **🎲 骰子图标**，选择生成模式。
4. **导入词库**：点击顶部工具栏的 **📥 图标**，支持批量粘贴或 JSON 文件。
