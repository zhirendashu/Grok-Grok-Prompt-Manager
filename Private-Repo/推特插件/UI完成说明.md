## ⚙️ 配置选项

### 默认配置
```javascript
{
    // 功能开关
    autoLike: false,        // 自动点赞（默认关闭）
    autoFollow: false,      // 自动关注（默认关闭）
    autoRetweet: false,     // 自动转发（默认关闭）
    originalsOnly: false,   // 只看原创（默认关闭）
    hdMedia: true,          // 高清图片（默认开启）

    // 速率限制
    rate: {
        like: 50,           // 每小时最多点赞 50 次
        follow: 20,         // 每小时最多关注 20 次
        retweet: 15         // 每小时最多转发 15 次
    },

    // 自动化配置
    config: {
        actionInterval: 2500,      // 操作间隔 2.5 秒
        maxActionsPerCycle: 2,     // 每次最多处理 2 条推文
        tickMs: 15000              // 检查间隔 15 秒
    }
}
```

### 修改配置
在控制台执行：
```javascript
// 修改速率限制
TwitterAssistant.state.rate.like = 10;  // 改为每小时 10 次

// 修改检查间隔
TwitterAssistant.state.config.tickMs = 30000;  // 改为 30 秒

// 修改操作间隔
TwitterAssistant.state.config.actionInterval = 5000;  // 改为 5 秒
```
