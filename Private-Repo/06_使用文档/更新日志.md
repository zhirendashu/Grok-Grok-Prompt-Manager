# 📅 更新日志 (Changelog)

本文档记录 **Grok Prompt Manager** 的所有版本更新历史。

---

## v5.0 (2026-01-18) - 里程碑版本
>
> **状态**: 正式发布

### 🚀 重大更新

1. **正式版发布**: 集成了之前测试版的所有功能，作为长期稳定版本。
2. **敏感词清洗**: 彻底替换了原有的敏感提示词范本，确保符合 GreasyFork 及社区规范。
3. **版权保护**: 添加了作者署名及防伪标识。

### ✨ 新增功能

- **📸 写真模式**: 一键生成专业人像提示词。
- **🎬 视频模式**: 独立的视频提示词库 + 随机生成逻辑。
- **🤖 自动重试**: 当网络错误时自动重试，支持 "Fix Prompt"。
- **🎨 UI 升级**: 全新的极简线条风格 (SVG) + 玻璃拟态设计。
